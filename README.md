# 🧠 인공와우 DSP 마스터 스터디 위키 (Cochlear Implant DSP Wiki)

본 학술 위키는 초저전력·초저지연 사양을 요구하는 인공와우(Cochlear Implant) 하드웨어 플랫폼을 타겟으로 하는 **실시간 디지털 신호처리(DSP)** 핵심 개념들을 고찰하고 체계적으로 정리한 정적 스터디 자료집입니다.

본 교재는 아날로그 음향 신호가 디지털 MEMS 마이크를 통해 고속 1비트 PDM 신호로 유입되어 수집되는 시점부터, 하드웨어 데시메이션을 거쳐 PCM 신호로 변환되고, 주파수 도메인 상에서 22개의 달팽이관 자극 전극 신호로 변환되기까지의 전 과정을 7개의 핵심 학습 모듈과 1개의 종합 마스터 커리큘럼 로드맵으로 나누어 구성하였습니다.

---

## 📂 단계별 학습 가이드 (7대 모듈)

선행 관계 및 학습 위계에 따라 아래의 모듈 순서대로 단계별로 학습하시는 것을 권장합니다. 각 모듈은 학술적인 수리적 모델 정의와 함께 C 및 Python 참조 구현을 포함하고 있습니다.

### [🎙️ Module 1. 디지털 마이크 & 데시메이션](01_dmic_pdm_decimation.md)
* 외부의 아날로그 음향이 디지털 MEMS 마이크(DMIC)를 통해 1비트 PDM 신호로 고속 오버샘플링되는 과정과, CIC 및 FIR 필터를 거치며 16kHz PCM 신호로 데시메이션되는 하드웨어 파이프라인의 수리적 연산 구조를 설명합니다.

### [🔢 Module 2. 24비트 Q1.23 고정소수점](02_fixed_point_q1_23.md)
* 저전력 DSP 임베디드 오디오 연산의 근간인 고정소수점(Fixed-Point) 산술의 수학적 정의를 다룹니다. Q1.23 비트 레이아웃, 곱셈 연산 후 23비트 우측 시프트(Right Shift)의 기하학적 의미, 56비트 누산기(Guard Bit Accumulator)를 활용한 오버플로우 방지 전략을 상술합니다.

### [📡 Module 3. 나이퀴스트 샘플링 & 폴딩](03_nyquist_folding.md)
* 16kHz 오디오 샘플링 체계에서 나이퀴스트 주파수(8kHz)의 물리적·수학적 한계를 분석하고, 고주파수 성분의 유입 시 발생하는 주파수 폴딩(Frequency Folding) 현상을 기하학적 반사선(Reflection Line) 모델과 공식으로 규명합니다.

### [🔂 Module 4. FIFO 버퍼, 지연 & 임펄스](04_fifo_latency_impulse.md)
* 1ms 홉(16샘플) 주기마다 512샘플 크기의 주파수 분석 버퍼를 실시간으로 갱신하는 링버퍼(Ring Buffer)와 비트 연산 마스킹(`& 511`) 최적화를 분석합니다. 아울러 FIR 필터의 군지연(Group Delay) 특성과 시스템의 고유 유전자인 임펄스 응답(Impulse Response)을 다룹니다.

### [🌊 Module 5. 과도기 윈도우 & FFT](05_fft_windowing.md)
* 시스템 초기 기동 후 유입 데이터가 512샘플 미만인 1~31ms의 초기 과도기(Transient Phase) 상태에서의 제로 패딩(Zero-Padding)의 영향과, Hann 윈도우 적용 시 발생하는 신호 소멸 왜곡을 방지하기 위한 초기 직사각형 윈도우 우회(Bypass) 전략을 논합니다.

### [🌀 Module 6. 복소수와 신호 분석](06_complex_numbers.md)
* 물리적 실수(Real) 소리 신호의 고유 에너지를 위상 변화와 무관하게 안정적으로 산출하기 위해 허수($j$)가 도입된 물리적 배경을 설명합니다. 2차원 회전 벡터 공간에서의 복소수 직교 좌표 평면을 시각화하고, 복소 크기(Magnitude) 및 위상(Phase)을 Q1.23 고정소수점 산술로 계산하는 효율적인 C 구현체를 제시합니다.

### [⚡ Module 7. RDFT 최적화 & 에너지 추출](07_rdft_energy.md)
* 실수 신호의 주파수 대칭성(Conjugate Symmetry)을 활용하여 512점 실수 데이터를 256점 복소 FFT 연산으로 수용하고 이를 Unpacking함으로써 클럭 및 메모리 점유율을 50% 절감하는 최적화 공식을 해부합니다. 더불어 257개 스펙트럼 빈(Bin) 데이터를 청각의 비선형 로그 척도에 맞추어 22개 전극 채널로 매핑(Binning)하는 밴드 패스 필터 구조를 구현합니다.
